Перед запуском изменить в параметр self.dir на свой путь до папки images/

Модуль content_analyzer.py - для обработки url статей с википедии.
Модуль tfidf_retrieval.py - считает tiff для текста и запроса и сортирует по sim расстоянию
Модель inverse_index_retrieval - генерирует индекс и может выдать два результата - OR и AND. Отличие в том, что AND выдаст только те номера документов, в которых встречаются все слова из запроса.
Модуль test.py - тестирование всех вышеописанных модулей и расчет метрик качества.
